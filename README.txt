# Tineco Label Generator V2

This is a static web app for GitHub Pages.

## Files to upload

Upload these files into your GitHub repository root:

- index.html
- style.css
- script.js

## GitHub Pages setup

1. Open your repository.
2. Go to Settings.
3. Go to Pages.
4. Source: Deploy from branch.
5. Branch: main.
6. Folder: /root.
7. Save.

Your website will be:

https://YOUR-GITHUB-USERNAME.github.io/tineco-label-generator-v2/

## Excel / CSV batch format

Use columns:

MODEL,SN,DATE

Example:

FW0704BOUS,TCN7YXNJJ7940,JUN-2026
FW0704BOUS,TCN7YXNJJ7941,JUN-2026

DATE is optional.

## Printing

Use:

- Paper size: 100 × 150 mm
- Orientation: Landscape
- Scale: 100%
- Margins: None / Default printer minimum

Use the calibration controls if the label is slightly shifted.
